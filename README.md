# exagent

A Python library for building agents faster. Provides base classes and utilities to create agents with support for tools, skills, and more.

> **Note:** This project is under active development. APIs may change between versions.

## Installation

```bash
pip install exagent
```

## Quick Start

```python
from exagent import Agent

agent = Agent()
agent.run()
```

## Features (Planned)

- Base `Agent` class for building custom agents
- Tool integration support
- Skill system for reusable agent capabilities
- Extensible architecture

## Requirements

- Python >= 3.9

## Development

This project uses [Poetry](https://python-poetry.org/) for dependency management.

```bash
# Clone the repository
git clone https://github.com/SachinDas246/exagent.git
cd exagent

# Install dependencies
poetry install
```
